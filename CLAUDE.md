## Run it

```bash
# 1) deps (first time)
python3 -m venv .venv && ./.venv/bin/pip install -r requirements.txt
# 2) backend (loads model artifacts, ~30s) — from repo root
./.venv/bin/uvicorn portal.server.main:app --port 8100
# 3) frontend (separate shell)
cd portal/client && npm install && npm run dev   # → http://localhost:5180
```

Tests: `./.venv/bin/pytest -q` (backend) · `cd portal/client && npx playwright test` (e2e,
auto-starts both servers). Models are gitignored `.pkl` artifacts regenerated by each
module's `train.py`; if missing, retrain (see `README.md`).

## Resuming / progress ledger

**Source of truth for progress is `program_state.json`** (task ledger); narrative is

phase per session (update ledger → append session-log entry → commit → STOP); keep that
cadence for any new phase.
