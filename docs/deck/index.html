Governance Signal — Two Genuine BLOCK Escalations

EWS and Line-Increase both surfaced noise-capped synthetic targets where subagents refused to game a mathematically-unreachable metric gate — escalating to a human decision instead of silently accepting bad numbers. That governance behavior is the point: the system flags when a spec is unachievable rather than papering over it.

Resumable Source of Truth

program_state.json is the task ledger — task status, next_action, and session checkpoints. Each session end: ledger updated, SESSION_LOG.md appended, commit tagged, resume command printed. The build can pause across daily token limits and resume exactly.
